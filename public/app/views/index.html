About This Page

This project was built using AngularJs and Bootstrap-CSS on the front-end and Node-JS and MongoDB on the backend for storing and retrieving data. Additonal technologies and libraries used include: JQuery, Mongoose, and Animate.CSS

The operations that can be performed by this application are:

Register new User

Register a new user into the System. Initially all user permissions are assigned as 'user' by default. An admin or moderator can update the permissions of all new users.

Update/Delete User details

An Admin or a moderator can edit the details of an existing user. A moderator can only edit details of users who are not 'admins'. Only admins can delete another user.

Change Permission level

A moderator can upgrade/downgrade Permission to 'user' and 'moderator'. An admin can upgrade/downgrade users of all permission levels.
